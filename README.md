# TouchDesigner Tools

A collection of TouchDesigner tools, tested in TouchDesigner 2021.16410 (stable).

![TouchDesigner Logo](https://github.com/CorrelateVisuals/Touchdesigner_Tools/blob/main/TouchDesigner_Logo.PNG?raw=true)

## About TouchDesigner

TouchDesigner is a node-based visual programming platform known for real-time rendering, audio-visual performance, and generative art. It scales well to large systems and allows low-level data manipulation through Python scripts, GLSL TOPs, and custom C++ operators.

**Download TouchDesigner:** [Derivative](https://derivative.ca/)

## Available Tools

| Tool | Description |
|------|-------------|
| [FITS File Scaling](./FITS_File_Scaling) | Load and scale FITS (Flexible Image Transport System) files from astronomical data |
| [Voronoi Diagram](./Voronoi_Diagram) | Generate Voronoi diagrams for spatial partitioning based on distance to points |
| [Image to Number of Points](./Image_To_Number_Of_Points) | Convert images to a specific number of 3D points (useful for drone shows) |
| [Directional Vector](./Directional_Vector) | Work with directional vectors (unit vectors) in 3D space |
| [SOP to Pointcloud](./SOP_To_Pointcloud) | Export SOP geometry data to EXR pointcloud format |
| [Hertzsprung-Russell Diagram](./Hertzsprung-Russell_Diagram) | Create stellar classification diagrams plotting luminosity vs. temperature |

## Getting Started

1. Download and install [TouchDesigner](https://derivative.ca/)
2. Clone this repository or download the tool you need
3. Open the `.toe` file in TouchDesigner
4. Each tool folder contains its own README with specific usage instructions

## Requirements

- TouchDesigner 2021.16410 (stable) or later
- Additional requirements vary by tool (see individual README files)
